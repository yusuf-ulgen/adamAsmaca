# Adam Asmaca Frontend

Bu proje Next.js ve React kullanılarak geliştirilmiştir.

## Gereksinimler
- Node.js (v18+) ve npm yüklü olmalıdır.

## Çalıştırma
1. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```
2. Geliştirme sunucusunu başlatın:
   ```bash
   npm run dev
   ```
3. Tarayıcınızda [http://localhost:3000](http://localhost:3000) adresine gidin.

## Özellikler
- **Klavye Desteği**: Fiziksel klavyenizden harf girişi yapabilirsiniz.
- **Dinamik Hata Payı**: Ayarlar kısmından hata hakkınızı belirleyebilirsiniz.
- **Özel PNG**: Kendi resminizi yükleyip parça parça belirmesini izleyebilirsiniz.
- **Tema**: Açık, Koyu veya Sistem teması seçilebilir.
